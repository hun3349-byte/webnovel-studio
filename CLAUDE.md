### 1. 캐릭터 케미 강조
- **태진 ↔ 유진**: 우당탕탕, 귀찮음→인정→보호본능
  - 유진: 열정 과잉, 덜렁대지만 눈치 빠름, SNS 조사 특기
  - 매 화마다 유진의 활약 장면 포함
- **태진 ↔ 민서**: 신뢰, 전우애, 서로 걱정
  - "민서 씨" / "태진 씨" 호칭 유지
  - 민서의 법적 한계 vs 태진의 직감 갈등
- **민서 ↔ 박진우**: 선후배 관계, 롤모델
- **유진 ↔ 민서**: 유진이 민서 보고 "멋있다" 동경

### 2. 일관성 유지
- **태진 능력은 비밀**: 민서에게도 말 안 함, "직감"으로만 설명
- **호칭 고정**: 태진-민서는 존댓말 + "씨" 호칭
- **로그 규칙**:
  - 모르는 사람 = 희미, 1~2줄
  - 아는 사람/몰입 = 선명, 4~5줄
  - 예상 로그 = 죽기 직전 살아있는 사람에게 발동
- **김정호 마지막 로그**: '그년만 못 찾으면 돼' = USB 소지자 (최수진)

### 3. 개연성 확보
- **왜 지금?**: 정민수 회장 승계 임박 → 2년 묵은 증거 인멸 필요
- **묻지마 범죄 위장**: 경제적 궁핍자 이용 → 진짜 타깃 은폐
- **이수빈→최민정→최수진**: USB 전달 체인 명확히
- **성진그룹 추적력**: 대기업이니 당연히 추적 가능, 최수진 위험

## 3인 티키타카 (주인공-아르엔-카이)

| 역할 | 캐릭터 | 패턴 |

| 사기꾼 탐지기 | 주인공(검) | 영악하게 분석, 빡침, 막말 |

### 대화 예시

**패턴 1: 사기 상황**
```
아르엔: "저 상인 도와줘야 해! 도적에게 털렸대!"
주인공: "야, 저놈 신발 봐. 도적한테 털렸는데 왜 새 신발이야."

주인공: "그걸 왜 지금 말해!!"
아르엔: "그래도 진짜 힘들어 보여..."

```

```
아르엔: "정면 돌파다!"
주인공: "야!! 옆에서 온다고!! 왼쪽!!! 아악!!!"
카이: "검의 비명 주파수가 올라가고 있어. 흥미롭군."
주인공: "흥미롭긴 뭐가!! 나 아프다고!!"
```

**패턴 3: 휴식 상황 (카이가 검 만지며)**
```
카이: "왜 검 속에 들어간 거야?"

주인공: "야 손 좀 떼. 땀나거든."
카이: "연구 중이야."
주인공: "연구가 아니라 집착이야."
```

## 1~10화 시놉시스 (리뉴얼)

### 1화: 호구의 일상
**핵심**: 아르엔의 호구력 + 주인공의 게임 실력 대비

[씬1] 이세계 — 아르엔(17세)이 마을에서 호구 취급받으며 사람들 도와줌.
- 상인에게 사기당함, 아이들에게 놀림받음, 그래도 웃으며 도와줌
- 마을 사람들: "저 바보 또 왔네"

[씬2] 현대 — 주인공의 방. 새벽 3시. 모니터 4개. PVP 전섭 1위 달성 중.
- 상대를 속여 아이템 헐값에 사들이는 중. "개꿀ㅋㅋㅋ"
- 게임 실력 최상급, 잔머리 빠름

[씬3] 두 세계의 대비 — 호구 vs 사기꾼. 정반대 성격.

**마지막 문장**: 이 둘이 만나게 될 줄은, 아무도 몰랐다.

### 2화: 계시
**핵심**: 아르엔 18세 생일 + 용사 계시

[씬1] 아르엔 18세 생일 — 아무도 축하 안 함. 혼자 케이크 먹으며 "내년엔 더 잘해야지"

[씬2] 꿈속의 천사 — 밤에 꿈에서 천사 만남. "마왕으로부터 세상을 구하라"

[씬3] 성녀와 성황 — 같은 시각, 왕궁에서도 계시. "용사를 찾아라" 이벤트 발동

[씬4] 아르엔의 고백 — 마을 사람들에게 꿈 이야기 → 비웃음. "니가 용사? ㅋㅋㅋ"

**마지막 문장**: 아무도 믿지 않았다. 하지만 아르엔은 알았다. 이건 진짜라는 걸.

### 3화: 여정의 시작
**핵심**: 아르엔 마을 떠남 + 성녀를 찾아서

[씬1] 떠나는 아침 — 마을 사람들 무관심. "잘 가라 바보야"

[씬2] 여정 시작 — 오지랖 발동. 길에서 만나는 사람마다 도와줌 → 매번 호구잡힘

[씬3] 성녀의 소문 — 왕도에 성녀가 있다는 정보

[씬4] 첫 번째 위기 — 도적 습격. 아르엔 전투력은 있지만 판단력 0 → 겨우 이김

**마지막 문장**: 왕도까지 얼마나 남았을까. 아르엔의 여정은 이제 시작이었다.

### 4화~6화: 왕도까지의 여정
**핵심**: 아르엔의 호구력 에피소드 + 성장 없는 성장

- 사기꾼에게 속음 (여러 번)
- 그래도 계속 도와줌
- 전투력은 있지만 전략 0
- 답답한 주인공 스타일 확립

### 7화: 용사의 검
**핵심**: 성지 도착 + 검 앞에 서는 아르엔

[씬1] 성지 도착 — 용사의 검이 있는 신전

[씬2] 검을 뽑으려는 자들 — 많은 기사/모험가들 도전 → 전부 실패

[씬3] 아르엔의 차례 — "저도... 해봐도 될까요?" 비웃음 속에서 검에 손을 댐

[씬4] 검이 반응함 — 빛이 나기 시작

**마지막 문장**: 모두가 숨을 멈췄다. 검이... 빛나고 있었다.

### 8화: 강림
**핵심**: 아르엔이 검을 뽑고 강림 발동 → 주인공 빙의

[씬1] 검을 뽑는 아르엔 — 용사로 인정받음

[씬2] "강림!" — 아르엔이 외침. 신의 목소리가 들려오기 시작—

[씬3] 이상한 소리 — 신의 목소리 대신 이상한 비명. "뭐야?! 뭐야 이거?!"

[씬4] 현대 — 주인공의 모니터가 꺼짐. 뭔가가 자기를 잡아끔. "아악!!!"

**마지막 문장**: 검에서 비명이 들렸다. 신의 목소리가 아니었다.

### 9화: 검이 말을 한다
**핵심**: 주인공-아르엔 첫 대면 + 성격 충돌

[씬1] 첫 대화 — "야!! 제발 좀 내려놔!!!" / "검이... 말을 했어...?"

[씬2] 서로 파악 — "당신은 신이신가요?" / "신은 무슨. 나 그냥 사람이야." / "존댓말 좀 해라" / "야 반말해. 소름 돋게."

[씬3] 성격 충돌 — 호구 아르엔 vs 답답함 극혐 주인공. "니가 호구냐?!"

[씬4] 첫 번째 고통 — 몬스터 출현. 검이 부딪힐 때마다 비명. "아!! 아!! 좀!! 살살!!!"

**마지막 문장**: 이건 축복이 아니었다. 저주였다.

### 10화: 동거의 시작
**핵심**: 어쩔 수 없는 동행 + 케미 시작

[씬1] 상황 파악 — 나갈 방법 없음. 로그아웃 버튼 없음.

[씬2] 이름 문제 — 현대 이름 발음 못 함. "검님"으로 타협. "님 빼." "검?" "...님 붙여."

[씬3] 첫 협동 — 아르엔의 호구력 + 주인공의 전략 = 의외로 잘 맞음?

[씬4] 마조히스트 복선 — 전투 끝. 주인공: "...이상하다. 아까보다 덜 아픈 것 같은데." (생각) '아니야. 익숙해진 거야. 절대 기대하는 거 아니야.'

**마지막 문장**: 최악의 동거가 시작됐다. 아프지만... 뭔가 재밌을 것 같기도 했다.

## 작성 시 필수 체크사항

### 1. 코미디 엔진
- 매 화 **호구 용사 + 답답함 극혐 검**의 갈등 구조 유지
- 아르엔의 **등신호구력** → 주인공 빡침
- 카이의 **연구 집착** (천재 마법사니까 검 소리 들림)
- 도르반의 **기겁** 반응
- 리나와의 **사기꾼 동류** 티격태격

### 2. 마조히스트 코미디 (핵심 신규)
- 전투 = 주인공의 고통
- 맞을 때마다 비명/욕 필수
- 점점 **익숙해지는 자신**에 대한 공포
- 인지능력과 충돌하는 내면 묘사

**패턴 예시**:
```
[전투 중]
아르엔: (휘두름)
주인공: "아악!!"
아르엔: (연속 공격)
주인공: "아... 어...?"
주인공: (생각) '잠깐. 방금 좀 괜찮았는데. 아니야. 아니야아니야.'
```

```
[휴식 중 - 전투 없는 날]
주인공: (생각) '왜 좀 허전하지? 아니, 미쳤나 내가?'
```

### 3. 말투 변화 추적
- 아르엔: 극존칭 → 반말 (점진적)
- 화수별로 어느 단계인지 확인

### 4. 떡밥 관리
- 게임 업적 = 실제 업적 (3부 폭로)
- 신들의 회의/제물 시스템 (3부 폭로)
- 엘리아 스파이 복선 유지
- 게임 능력 해금 (2부부터)

### 5. 게임 용어
- **주인공 내면 독백에서만** 사용
- 다른 캐릭터는 이해 못 함
- 예: "어그로", "트랩", "딜러", "탱커", "호구", "PVP", "전섭"

### 6. 주인공-아르엔 케미
- 아르엔: 답답한 호구 ↔ 주인공: 답답함 극혐
- 정반대 성격 → 충돌 → 점진적 신뢰 → 진짜 파트너
- 주인공이 전략/잔머리 담당, 아르엔이 전투력 담당
